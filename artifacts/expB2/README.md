# Experiment B-v2 published artifacts

Canonical committed outputs from pre-registered confirmatory runs (not ephemeral
`results/runs/` bundles from `scripts/run_e2e.py`).

| File | Description |
|------|-------------|
| `expB2_results.json` | Default n=3 confirmatory metrics |
| `expB2_survival.png` | Default n=3 figure |
| `expB2_results_confirmatory_n3.json` | Named copy of confirmatory sweep |
| `expB2_survival_confirmatory_n3.png` | Named confirmatory figure |

New runs write to `results/runs/<timestamp>/artifacts/` or here when promoted manually.
